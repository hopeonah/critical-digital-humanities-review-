TOOL EXPLORATION
The tool exploration for AntConc would be a free toolkit analysis that was designed by the owner of AntConc, and it is used for corpus linguistics and DH studies. The accessing of the tool was achieved by visiting the website directly by the developer and a dowloading tool to a personal computer. The programme was created so that the user account cannot be created and connected to a server or the internet after the programme has been installed, allowing it to be accessible for the end users. The installation has been displayed through the downloadable files with versions that are available for Linux, Windows, and Mac. (Anthony, 2019)
AntConc provides a GUI, this is organized by clearly visualising the tabs for word list, collocates amd clusters, and other analysis. This enables easy flow and transitions between different types of analysis due to the structure of the layout by structuring them into different tabs, each having specific needs, making it easier for first-time users and for use its easy to navigate in terms of the basics, such as the regular expressions and keyword analysis referencing a reference corpus. 
The main Function of AntConc as a tool may be described as a clear, consistent analysis, especially when carried out from a corpus perspective. This tool allows researchers to analyze contextual word use, which is useful for DH studies research. There are a lot if positives with this app, but it also comes with its negatives, such as the software only allows plain text files, limited data vizualization and it isn't easy to annotate. AntConc has a minimal impact since data analysis is carried out locally; its freeware model aligns well with renewable perspectives with DH. (AntConc, 2026)

COMPARITIVE ANALYSIS 
AntConc can be compared to a lot of other tools, such as TAPoR and Voyant tools. Voyant Tools is a web-based reading and analysis environment for digital tools (Voyant Tools, n.d). This tool has been noted by Sinclair and Rockwell. The advantages of AntConc lie in the ability to operate the computer with access to the internet, compared to Voyant Tools, which has the ability to work without the need for any uploading, which can cause disruption for other people's data, such as uploading copyrighted content. 
  (Anon., n.d.) – Voyant tools image

TAPoR is used for various text analysis programs that allow users to carry out different responsibilities, which are repetition count and linguistic patterns (Radzikowska and Rockwell, n.d.), compared to AntConc, which can be installed on the computer. TAPoR depends on the user to connect to the internet and upload the data to the server for further access later on. TAPoR users are able to choose different variations of programs for different tasks, but the only disadvantage is that the detailed control settings for each program are not supported by the software itself. AntConc pros are transparency, consistency, and local processing, which secures the verification and safety of sensitive information. With all the advantages of AntConc has visualization capability and flexibility perform tasks online. TAPoR and Voyant are more practical, providing more visualization and flexibility, but are useful only when an internet connection is present. 
Collectively, the selection of tools relies on research goals. AntConc would be more direct to corpus research, Voyant is more for visual forms of research, and lastly, TAPoR is best for multi-tool-based research. Each research demands clear linguistic research.
